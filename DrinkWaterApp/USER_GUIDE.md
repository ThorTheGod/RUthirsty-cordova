# 💧 喝水打卡应用 - 完整使用指南

## 🎯 应用概览

这是一个带有二次元少女提醒助手的喝水打卡应用，采用毛玻璃（Glassmorphism）设计风格，帮助你养成健康的喝水习惯。

---

## 📱 主要功能

### 1. 喝水打卡
- **打卡按钮**: 点击中央的"打卡喝水"按钮记录喝水
- **今日统计**: 实时显示今天喝水的次数
- **总计统计**: 显示累计喝水总次数
- **记录列表**: 查看所有喝水记录，包括时间和日期

### 2. 二次元少女助手 🎀
- **位置**: 屏幕右下角
- **外观**: 粉色脸蛋、紫色头发、会眨眼微笑
- **动画**: 持续浮动、眨眼、眼珠移动、头发摆动

#### 互动方式
- **点击角色**: 显示鼓励语和今日喝水次数
- **自动提醒**: 定时弹出聊天气泡提醒喝水
- **点击气泡**: 关闭提醒气泡

### 3. 智能提醒系统 ⏰
- **自动提醒**: 根据设定间隔自动提醒
- **随机文案**: 15条可爱提醒语随机显示
- **欢迎提醒**: 首次使用显示欢迎语
- **持久化**: 设置自动保存，刷新不丢失

### 4. 设置面板 ⚙️
- **提醒开关**: 启用/禁用定时提醒
- **间隔设置**: 5/15/30/60/120分钟可选
- **测试功能**: 立即测试提醒效果

## 🎮 操作指南

### 基础操作

#### 打卡喝水
1. 点击中央的"💧 打卡喝水"按钮
2. 记录自动添加到列表顶部
3. 今日次数和总计次数自动更新

#### 查看记录
- 向下滚动查看所有喝水记录
- 每条记录显示具体时间和日期
- 点击"删除"按钮可删除单条记录

#### 清空记录
1. 点击"清空记录"按钮
2. 确认后清空所有历史记录
3. 统计数据重置为0

### 提醒功能

#### 打开设置
1. 点击记录区域右上角的"⚙️"按钮
2. 设置面板从中央弹出

#### 调整提醒间隔
1. 在设置面板中选择间隔时间
2. 点击对应按钮（5/15/30/60/120分钟）
3. 当前选中的按钮会高亮显示
4. 设置立即生效并自动保存

#### 开关提醒
1. 在设置面板中切换"启用定时提醒"开关
2. 关闭后不再自动提醒
3. 开启后恢复定时提醒

#### 测试提醒
1. 在设置面板点击"测试提醒"按钮
2. 立即显示一条随机提醒
3. 设置面板自动关闭

#### 关闭设置
- 点击右上角"✕"按钮
- 或点击面板外的背景区域

### 角色互动

#### 查看鼓励语
1. 点击右下角的二次元少女角色
2. 显示随机鼓励语或今日喝水次数
3. 再次点击或等待3秒自动关闭

#### 关闭提醒气泡
- 点击气泡任意位置
- 或等待自动消失（提醒5秒，手动3秒）

## 💾 数据存储

### 本地存储
应用使用浏览器的 localStorage 存储数据：

- **喝水记录**: `drinkWaterRecords`
- **提醒设置**: `reminderSettings`

### 数据持久化
- 所有数据自动保存
- 刷新页面不会丢失
- 清除浏览器数据会重置

### 数据格式

#### 喝水记录
```javascript
{
    id: 1738753200000,           // 唯一ID（时间戳）
    timestamp: 1738753200000,    // 记录时间戳
    date: "2026-02-05",          // 日期
    time: "14:30:00"             // 时间
}
```

#### 提醒设置
```javascript
{
    enabled: true,               // 是否启用
    interval: 30,                // 间隔（分钟）
    lastReminder: 1738753200000  // 上次提醒时间戳
}
```

## 📊 使用建议

### 健康喝水习惯
- **推荐频率**: 每30-60分钟喝一次水
- **每日目标**: 8-10次（约2000ml）
- **最佳时间**: 起床后、饭前、运动后

### 应用使用技巧
1. **设置合适间隔**: 根据作息调整提醒间隔
2. **及时打卡**: 喝水后立即记录
3. **查看统计**: 每天查看今日次数
4. **互动角色**: 点击角色获得鼓励

### 性能优化
- **定期清理**: 记录过多时清空旧记录
- **关闭提醒**: 不需要时可临时关闭
- **浏览器缓存**: 定期清理浏览器缓存

## 🔧 故障排除

### 提醒不工作
1. 检查设置面板中提醒是否开启
2. 确认浏览器允许通知
3. 检查浏览器控制台是否有错误

### 数据丢失
1. 检查是否清除了浏览器数据
2. 确认使用同一浏览器和设备
3. 避免使用隐私/无痕模式

### 动画卡顿
1. 关闭其他占用资源的标签页
2. 更新浏览器到最新版本
3. 检查设备性能

### 样式异常
1. 刷新页面（Ctrl+F5 强制刷新）
2. 清除浏览器缓存
3. 检查浏览器是否支持 backdrop-filter
