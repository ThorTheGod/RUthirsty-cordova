## 🚀 如何使用

### 方法 1: 浏览器测试（推荐）

**应用正在运行中！**

访问地址: **http://localhost:8000**

在 GitHub Codespace 中:
1. 点击底部的 "端口" 标签
2. 找到端口 8000
3. 点击 "在浏览器中打开" 🌐

### 方法 2: 使用构建脚本

```bash
cd /workspaces/RUthirsty-cordova/DrinkWaterApp
./build.sh
```

选择选项:
- `1` - 构建调试版 APK
- `3` - 在浏览器中测试
- `4` - 在设备上运行

### 方法 3: 手动构建

```bash
# 构建 Android APK
cordova build android

# APK 位置
platforms/android/app/build/outputs/apk/debug/app-debug.apk
```

## 🔧 配置选项

### 修改应用名称
编辑 `config.xml`:
```xml
<name>喝水打卡</name>
<description>健康喝水提醒应用</description>
```

### 修改颜色主题
编辑 `www/css/index.css`:
```css
/* 背景渐变 */
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

/* 按钮颜色 */
.drink-button {
    background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
}
```

### 修改存储键名
编辑 `www/js/index.js`:
```javascript
const STORAGE_KEY = 'drinkWaterRecords';
```
